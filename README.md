# csv-table-editor

A tool that accepts a CSV files and generates a table  that displays the data. The table should be editable and the user should be able to download the edited version of the CSV..

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/csv-table-editor.git
cd csv-table-editor
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
